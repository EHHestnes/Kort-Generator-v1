Programvaren er utviklet for folk som sliter med å skrive kort
i safeprogramet til Oddfjell.
Programet er gratis og tilgjenlig for vidreutvikling ved egent ønske
om det.
NetBeans IDE 8.2 for windows ble bruk for å skape programet.
Er 169 alternativ + mulighten for å legge til mer
Praktisk informasjon angående programmet så må mappen ligge helt
øverst i D disken altså D:\ for at det skal klare å finne ut hvor
bilde,kort.txt og listen er plassert.
programmet er desvere windows bassert selv om meste parten av koden vil 
fungre på en mac eller linux.
Skulle en få problemer med å få opp navner kort.text eller bilde så kopier
mappa Kortautomatotext mappen inn på D disken så kan en flytte programmer
program mappa over enn annen plass.
